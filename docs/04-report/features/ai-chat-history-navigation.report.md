## 4. 사용자 가이드

1. AI 채팅창에서 메시지를 한 번 이상 전송합니다.
2. 입력창이 비어있거나 무언가 입력 중인 상태에서 **위쪽 화살표(↑)**를 누릅니다.
3. 이전에 보냈던 메시지가 나타납니다. 다시 누르면 그 이전 메시지로 이동합니다.
4. 다시 **아래쪽 화살표(↓)**를 누르면 최신 메시지로 돌아오며, 마지막엔 원래 입력 중이던 텍스트가 나타납니다.
5. 여러 줄을 입력한 경우, 커서가 가장 윗줄에 있을 때만 위쪽 탐색이 작동하므로 일반적인 편집 동작을 방해하지 않습니다.
