# Report: ai-panel-auto-collapse

> **Feature**: AI 패널 자동 접힘 (외부 클릭 연동)  
> **Created**: 2026-05-13  
> **Phase**: Report  

---

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | AI 패널이 열려 있을 때 캘린더나 다른 UI 요소를 조작하려 할 때 매번 패널을 수동으로 접어야 함 |
| **Solution** | 브라우저 전역 클릭 감지(Click Outside)를 통해 패널 외부 클릭 시 자동 최소화 기능 구현 |
| **Functional UX Effect** | 사용자 관심사가 패널 밖으로 옮겨가면 자동으로 화면 공간을 확보해주는 지능적인 UX 제공 |
| **Core Value** | 불필요한 클릭 횟수 감소 및 사용자 흐름 유지 |

- [x] **예외 처리**: 패널 내부 및 헤더 클릭 시에는 기존 기능을 유지하고 닫히지 않도록 보호

## 4. 최종 사용자 가이드

1. AI 어시스턴트 패널을 엽니다.
2. 캘린더, 상단 네비게이션, 사이드바 등 **패널 외부의 어느 곳이든** 클릭해 봅니다.
3. 클릭 즉시 AI 패널이 자동으로 아래로 내려가며 캘린더 공간이 최대로 확장됩니다.
4. 패널 내부(메시지 리스트, 입력창)를 클릭할 때는 패널이 유지되므로 안심하고 조작하셔도 됩니다.
