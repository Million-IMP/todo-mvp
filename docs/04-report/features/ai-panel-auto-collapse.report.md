## 4. 사용자 가이드

1. AI 어시스턴트 패널을 열고 대화를 진행합니다.
2. 캘린더의 특정 날짜나 시간 슬롯을 클릭하거나, 이미 등록된 일정을 클릭해 봅니다.
3. 클릭과 동시에 AI 패널이 자동으로 아래로 내려가며 캘린더 영역이 넓어지는 것을 확인합니다.
4. 필요한 조작(일정 생성/수정 등)이 끝나면 하단 바를 다시 클릭하여 AI와 대화를 이어나갈 수 있습니다.
