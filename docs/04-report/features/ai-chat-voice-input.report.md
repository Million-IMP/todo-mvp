## 4. 사용자 가이드

1. AI 채팅 입력창 우측(전송 버튼 옆)의 **마이크 아이콘(🎤)**을 클릭합니다.
2. 버튼이 빨간색으로 변하고 깜빡이면 음성 인식이 시작된 것입니다.
3. "내일 오후 3시에 회의 일정 추가해줘"와 같이 한국어로 말합니다.
4. 입력창에 텍스트가 실시간으로 입력되는 것을 확인합니다.
5. 입력이 완료되면 마이크 버튼을 다시 눌러 중단하거나, 바로 전송 버튼을 눌러 AI에게 요청합니다.
