## 2. 세부 구현 사항 확인

### 2.1 UI 구성
- **아이콘**: 클립보드 체크 모양 아이콘 사용
- **리스트**: `subtasks.map`을 통한 렌더링, `group-hover`를 이용한 삭제 버튼 노출
- **입력**: Enter 키 지원 및 추가 버튼 배치

### 2.2 로직
- `uuidv4` 라이브러리를 통해 클라이언트 사이드에서 고유 ID 생성
- `subtasks` 로컬 상태를 통해 모달 내 편집 내용 관리
- 부모 컴포넌트(`onSave`)로 최종 상태 전달
